# Mapping verso Open Referral HSDS (minimo)
- `organization` → HSDS `organization`
- `service` → HSDS `service`
- `location` → HSDS `location`
- `channel` → HSDS `contact`/`phone`/`service_at_location` (a seconda dei casi)
- `taxonomy` → HSDS `taxonomy`/`service_taxonomy`

**Note**: questo repo è dimostrativo. L’aderenza completa a HSDS richiede tabelle aggiuntive (`phone`, `address`, `service_at_location`, ecc.).
